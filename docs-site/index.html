Why FlAI?

Own Your Code

Components are source-installed into your project. No package dependencies, no version conflicts. Customize everything.

AI-First Design

Purpose-built for AI chat: streaming text, thinking indicators, tool calls, citations, and multi-model support out of the box.

4 Theme Presets

Light, Dark, iOS, and Premium (Linear-inspired). Full design token system with colors, typography, spacing, and radii.

Zero Dependencies

Core components use only Flutter SDK. No state management libraries, no vendor lock-in. Vanilla ChangeNotifier + Streams.
